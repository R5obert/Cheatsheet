HTML Tag Reference

HTML Tags

Tag Name Description
<a> Anchor Is used to create hyperlinks, which allow users to navigate from one page to another.
<img> Image Image Element embeds an image into the document.
<link> Link Links a CSS file or any other kind of font or resource.

HTML Attributes

Attribute Name Description
href HyperText Reference It specifies the URL of the page or resource that the hyperlink points to.
src Source It is used to specify the URL of the external resource that an HTML element is referring to.
alt Alternative Text It provides a textual description of an image.
